# База данных проката автомобилей
Разрабатываемая подсистема носит название "БД проката автомобилей" и предназначена для:
* автоматизации обработки информации о клиентах, бронировании
и аренды;
* обеспечения возможности взаимодействия с интересущими пользователя клиентами, бронировании и арендами;
* обеспечения возможности оперативного получения инфомации о всех клиентов, бронирований и арендах.

В соответствии с предметной областью система строится с учётом следующих особенностей:
* клиент может бронировать несколько автомобилей;
* клиент может арендовать несколько автомобилей;
* каждый автомобиль бронируется/арендуется только одним клиентом;
